- ООП (Классы, Методы, Интерфейсы, Наследование, Полиморфизм).
- Работа с файлами.
- Мини проект.
- Более сложные программы. Работа с коллекциями (ArrayList, HashSet).
- Enum, рекурсия, двоичное дерево, работа с исключениями.
